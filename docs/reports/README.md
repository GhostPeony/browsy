# Reports

This folder stores benchmark outputs and summaries.

Typical workflow:
1) Run `benchmark/loop.ps1` (or individual benchmark scripts).
2) Review the generated CSV/JSON files and summary markdown.

Files:
- `YYYY-MM-DD-browsy-parallel-parse-iN.csv/json`
- `YYYY-MM-DD-browsy-parallel-fetch-iN.csv/json`
- `YYYY-MM-DD-agent-browser-parallel-iN.csv/json`
- `YYYY-MM-DD-playwright-parallel-iN.csv/json`
- `YYYY-MM-DD-benchmark-summary.md`
- `YYYY-MM-DD-benchmark-summary.csv`

Notes:
- Use `benchmark/manifest.json` to configure iterations, URLs, and thresholds.
- Summaries are generated by `benchmark/report.ps1`.
